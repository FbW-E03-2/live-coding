position - relative

An element with position: relative is positioned RELATIVE to it's normal position.

This div has position: relative;

position - absolute

An element with position: absolute is positioned relative to it's NEAREST POSITIONED ANCESTOR

this div element has position: relative;
this div element has position: absolute;

position - sticky

Allows a positioned element to act like it's relatively positioned UNTIL it is scrolled to a certain threshold point (eg 10px from the top of the viewport), after which it becomes fixed.
