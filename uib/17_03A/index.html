Ordered and Unordered Lists

Ordered Lists are used when things need to happen in a particular order

  1. wake up
  2. make espresso
  3. wash face
  4. prepared for class
  5. started coding!

Unordered lists are for when order is not important

  • mineral water
  • pretzels
  • tequila
  • coffee
  • gummi bears

Nesting Lists

  • Item 1

    1. Subitem A
    2. Subitem B

  • Item 2

    • Subitem A
    • Subitem B
